# Employee Directory

A Django-based Employee Management System with CRUD operations.

## Features
- Add, update, delete employees
- Admin panel
- Media upload support

## Tech Stack
- Python
- Django
- SQLite

## How to Run
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Run server
